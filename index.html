Key Modules

Core components that compose a demo environment

Location Engine

GPS-first resolution, IP fallback, device metadata & accuracy data for richer context.

Installation

Quickstart

Clone the demo repo and run the safe installer:

git clone https://github.com/Bibekbista0406/Looking_You/tree/main
cd looking-you
chmod +x look-kri.sh
./look-kri.sh
Quick checklist
  • Confirm you have written authorization for testing
  • Choose tunnel provider (Cloudflare, ngrok, SSH)
  • Review retention & logging options
